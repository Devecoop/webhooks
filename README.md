# Webhooks

Realtime HTTP notification from Lelylan to third party apps

## Requirements

Webhooks is tested against Node 0.10.36.


## Installation

    $ git clone git@github.com:lelylan/webhooks.git && cd webhooks
    $ npm install && npm install -g foreman
    $ nf start
