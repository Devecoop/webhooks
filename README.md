# Webhooks

Webhooks dispatcher to enable third party updates with Lelyan.


## Requirements

Webhooks is tested against Node 0.8.8.


## Installation

    $ mongod
    $ git clone git@github.com:lelylan/webhooks.git
    $ npm install && npm install -g foreman
    $ nf start
